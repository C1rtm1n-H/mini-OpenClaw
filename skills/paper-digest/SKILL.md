# 论文速读

1. 用 `glob` 定位 PDF、Markdown 或文本文件；用 `read` 读取文本。若只有 PDF，先用 `bash` 调用环境中已有的 PDF 文本提取工具，再读取提取结果。
2. 按“研究问题、核心贡献、方法、数据与实验、主要结论、局限性”提取证据；为关键结论保留页码、章节或原文位置。
3. 区分作者明确陈述与自己的推断；无法从材料确认的内容标为“未说明”。
4. 输出结构化速读报告，并给出值得复查的图表、公式或实验设置。

## 注意事项

- 论文解析失败时说明受限原因，并请求文本版或可复制内容。

## 示例输出

```markdown
# 论文速读：<题名>
- 研究问题：...
- 核心贡献：...
- 方法：...
- 数据与指标：...
- 主要结论：...（第 4.2 节）
- 局限性：...
- 待复查：图 3 的消融设置未说明随机种子。
```
